A simple plug-in to configure plain text post for user roles and post types.

== Description ==

WP Plain Text Post allows to configure plain text post or post with allowed HTML tags by post types and by user roles.

List of features:

* Allows to configure plain post input with no HTML tags.
* Allows to configure the post with allowed HTML tags.
* Allows to configure by user roles.
* Allows to configure by post types.

== Installation ==

1. Upload `wpplainpost` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin via Settings -> WP Plain Post
